####  快速幂公式









####  组合数公式1

$$
C_a^b = C_{a-1}^{b-1} + C_{a-1}^b
$$

公式解释



从 a 个苹果 里面 选出 b个苹果，

1. 选出一个绿色苹果， -> 剩下的方案是 $C_{a-1}^{b-1}$
2. 不选这个苹果- > 剩下的方案是 $C_{a-1}^{b}$ 

根据加法原理： $C_a^b = C_{a-1}^{b-1} + C_{a-1}^b$



```cpp
 //预处理
    for(int i=0;i<=2000;i++)
    {
        for(int j=0;j<=i;j++)
        {
            if(!j) f[i][j]=1;
            else f[i][j]=(f[i-1][j-1]+f[i-1][j])%mod;
        }
    }
```
