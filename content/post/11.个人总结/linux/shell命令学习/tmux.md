## 使用快捷键

- **左右分屏** ：Ctrl + b, % (分割当前窗口)
- **上下分屏** ：Ctrl + b, " (分割当前窗口)
- **关闭分屏** ：Ctrl + b, x (关闭所在分屏窗口)
- **显示分屏编号** ：Ctrl + b, q (显示分屏编号)
- **分屏切换** ：Ctrl + b, 方向键 (基本可以自由切换)
- **关闭当前** **panel**:  ctrl + d
