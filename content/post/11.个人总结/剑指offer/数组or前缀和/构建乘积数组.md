给定一个数组`A[0, 1, …, n-1]`，请构建一个数组`B[0, 1, …, n-1]`，其中B中的元素`B[i]=A[0]×A[1]×… ×A[i-1]×A[i+1]×…×A[n-1]`。

不能使用除法。

#### 样例

```
输入：[1, 2, 3, 4, 5]

输出：[120, 60, 40, 30, 24]
```

**思考题**：

- 能不能只使用常数空间？（除了输出的数组之外





```cpp
class Solution {
public:
    vector<int> multiply(const vector<int>& A) {
        int n = A.size();
        if(n<=0) return {};
        vector<int> v(n,1);
        for(int i=1,p=A[0];i<n;++i) {
            v[i] = p;
            p*= A[i];
        }
        for(int i=n-1,p=1;i>=0 ;--i) {
            v[i] *= p;
            p *= A[i];
        }
        return v;
    }
};
```
