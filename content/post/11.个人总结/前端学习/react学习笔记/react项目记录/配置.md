##  react 代码配置笔记


next 安装 ant-design 过程

```shell
yarn add @zeit/next-css
# 安装 可以在 react 上写 css的框架
yarn add antd

# 安装 ant-design


yarn add babel-plugin-import

# 安装 babel 按需加载
```

```json
{
    "presets": [
        "next/babel"
    ],
    "plugins": [
        [
            "import",
            {
                "libraryName": "antd"
            }
        ]
    ]
}

```
