<?php
    $os = array("Mac", "NT", "Irix", "Linux");
    if (in_array("Irix", $os)) {
        echo "Got Irix";
    }
    if (in_array("mac", $os)) {
        echo "Got mac";
    }
?>

如何子类函数调用基类方法, 在这个方法中显示子类的类名 ?

    1. 首先想到的是PHP的魔术常量 _CLASS_

    但是, 如果在基类中使用_CLASS_ 代表的是这个基类的类名, 而非调用它的子类的类名

    <?php
class Base{
    public test(){
        echo __CLASS__;
    }
}
class myTest extends Base{
    public my_test(){
        $this->test(); 	// 此时输出的为 Base 而非 myTest
    }
}

    <?php
class A
{
    function __construct()
    {
        echo get_class($this);	# 注意用get_class 需要传入$this
    }

    static function name()
    {
        echo get_called_class();
    }
}

class B extends A
{
}

$objB = new B(); // 输出 B
B::name();       // 输出 B
