总结

语言要设置为 mermaid

表述 说明 含义
id[文字] 矩形节点 表示过程，流程中一个环节
id(文字) 圆角矩形节点 表示开始和结束
id((文字)) 原型节点 表示连接
id{文字} 菱形节点 表示判断 决策
id>文字] 旗帜状节点
TB TOP to bottom 从上到下

类型 描述
-> 无箭头的实线
–> 无箭头的虚线
-» 有箭头的实线（主动发出消息）
–-» 有箭头的虚线（响应）
-x 末端为叉的实线（表示异步）
–x 末端为叉的虚线（表示异步）
