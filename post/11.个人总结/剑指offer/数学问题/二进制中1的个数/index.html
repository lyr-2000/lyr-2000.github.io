二进制中1的个数

2021-08-17

解题思路

$n&(n-1) 的结果为 110&101 ,得到 100,$

$n&(n-1),可以去掉一个1$

class Solution {
public:
    int NumberOf1(int n) {
        int res = 0;
        while(n) {
            res++;
            n = n&(n-1);
        }
        return res;
    }
};
