Golang 刷题入门_字符串中数字的个数

文章目录

输入一行字符，长度不超过 100100，请你统计一下其中的数字字符的个数。

输入格式

输入一行字符。注意其中可能包含空格。

输出格式

输出一个整数，表示字数字字符的个数。

1

package main

import (
	"bufio"
	"fmt"
	"os"
)

func main() {

	var input = bufio.NewReader(os.Stdin)

	var str, _ = input.ReadString('\n')
    // if err != nil {
    //     fmt.Println(err)
    //     // err == io.EOF
    // }

	// fmt.Println(str)
	//
	var cnt int
	for _, ch := range str {
		if ch >= '0' && ch <= '9' {
			cnt++
		}
	}
	fmt.Printf("%d\n", cnt)

}
