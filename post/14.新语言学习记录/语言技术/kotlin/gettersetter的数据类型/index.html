kotlin 的数据类

数据类型是 final的， 并且 自动生成 getter,setter方法，自带 toString 和 hashCode

数据类是 final类型的，不能再用 open 修饰。

data class User(var id: Int,var name:String )


fun main(args: Array<String>) {
//
    var user = User(18,"dog")
    println(user)


}

kotlin的枚举类型

kotlin解构运算符


class User(var id: Int,var name:String ) {
    operator fun component1() = id
    operator fun component2() = name

}


fun main(args: Array<String>) {
//
    var user = User(18,"dog")
    println(user)
    var (id, name ) = user
    print(id)

}




fun main(args: Array<String>) {
//
    var mp = mapOf<String,String>("key" to "value")

    for ((k,v) in mp) {
        println("$k = $v")
    }

}

getter和 setter语法


class Hello {
    var string:String? = null
    get() {
        return field + "get"
    }
    set(value) {
        field = value + "set"
    }
}
fun main(args: Array<String>) {
//    println(1 and 2)
    var h = Hello()
    h.string = "hello world"
    println(h.string)
}
