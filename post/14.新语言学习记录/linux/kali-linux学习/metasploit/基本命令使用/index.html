db is down 怎么解决

参考博客

sudo service postgresql start
# 记得加 sudo 权限
msfconsole
# service postgresql status

db_connect postgres:toor@127.0.0.1/msf
# 手动连接 postgresql
msf6 > db_status
[*] Connected to msf3. Connection type: postgresql.


apt-get install metasploit-framework -y
# 重新安装
