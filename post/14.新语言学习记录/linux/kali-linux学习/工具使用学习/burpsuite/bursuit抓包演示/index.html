模块总结

tab模块 用法
dashboard 仪表任务模块
target 目标请求响应全部记录
proxy 代理，可以查看拦截的信息
intruder 入侵自动化 【自动设置参数并请求】
repeater 请求重放，可以发送 http请求等 【类似 postman 那种东西】
sequencer 信息截取
decoder 解码编码
comparer 对比工具 【对比多次请求的返回结果】
logger 日志模块
extender 插件模块
project options 项目配置
user options 用户自定义配置
learn 官方文档指引

新建代理

点击 proxy tab栏

firefox 浏览器可以安装 foxproxy 插件， 然后用于设置代理

  1. firefox 点击证书

  2. 点击 options , export ca certificate ，点击 export der格式证书

  3. firefox 导入 burpsuite 证书

  4. 点击 proxy> intercept 点击 按钮 到 intercept is off ,

  5. 然后就可以请求了，所有的请求都经过 burpsuite 代理
